comandos básicos no git bash

cd pasta
-> entra na pasta

cd ..
-> retorna um nível

git init
-> inicia repositorio do git na pasta

git status
-> consultar se há alguma modificação no repositorio

git add <arquivo> ou *
->adiciona a(s) modificação para ser comitada
-> envia para stage

git commit -m "mensagem"
-> comita as modificações e coloca uma flag

git config --list
-> lista as configurações do git

git config --global --unset user.name
-> apaga o item name das configurações de usuario

git config --global user.nickname "nickname"
-> altera o valor da configuração nickname

->
criar repositório na pagina do git
copiar o caminho do repositorio
apontar o repositorio remoto, 
pois as alterações ainda estão no repositorio local
------
git remote add origin link ou git remote add origin/main link
git remote -v
-> retorna lista de repositorios remotos cadastrados


git push origin master ou só git push
->envia o codigo/modificações para o repositorio remoto



->se o repositorio remoto estiver com alguma alteração que 
vc não tenha no repositorio local, é necessário baixar as 
modificações antes para depois subir as suas modificações
-------
git pull origin master



git clone link
-> clona o projeto do repositorio remoto para o repositorio local
